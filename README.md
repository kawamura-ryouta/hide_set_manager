# Hide Set Manager

Blenderで編集途中に隠した頂点・面・オブジェクトの状態を、
「非表示セット」として保存・切り替えできる管理用アドオンです。

Blenderでは一時的な非表示操作が多用されますが、
後から同じ状態を再現することが難しいという課題があります。
本アドオンは、その非表示状態をセットとして記録し、
作業の再現性と管理性を向上させます。

---

## 主な機能
- 非表示状態をセットとして登録・管理
- 表示 / 非表示 / トグルの一括切り替え
- 編集モード / オブジェクトモード両対応
- 永続IDによる安全な要素追跡
- 差分同期（現在の状態との差分を検出して更新）
- JSONエクスポートによる外部保存

---

## 想定利用シーン
- 複雑なモデル制作中の表示管理
- 作業工程ごとの表示状態の切り替え
- 後から編集を再開する際の状態復元
- 非表示要素を含む作業の再現・共有
